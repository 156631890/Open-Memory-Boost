# Release Notes

## 0.2.0

- Added a richer memory engine with ids, tags, priority, status, update, forget, compact, stats, export, import, and summary workflows.
- Added a Python developer API for programmatic integrations.
- Added fuzzy compaction for near-duplicate entries.
- Added JSON export/import support.
- Added unit tests and broader CI coverage.

## 0.1.0

- First open-source release.
- Added Open Memory Boost skill.
- Added local Markdown memory runtime.
- Added GitHub Actions CI.
- Added English and Chinese project homepages.
